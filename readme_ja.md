# 日本国内におけるギャングストーキングについて

## 手口
ターゲットに統合失調症の患者が訴える病状を真似た嫌がらせをして
保護、措置診察等を経て精神病患者に仕立てあげる。

実際の病気と大きく違うところは嫌がらせをする人たち自体、
病気と思わせる気が全くないこと。
妙な書き方かもしれないが、経験者は納得できる表現のはず。

### 言葉、しぐさ系
悪口や個人的秘密のほのめかし、特定のしぐさや挑発の繰り返しを
場所を問わず何年も何千回、何万回と行う。（数は誇張ではない）
が、決して挑発に乗ってはならない。

### 機材系
個人的な検索をしていない会社でネットの広告が不自然に偏る。
「黄色い救急車」を連想させるような車や緊急車両などが
不自然なほど頻繁に近隣を走り回るようになる。
これも挑発に乗ってはならない。

### 化学物質系
自宅の換気機能が押さえられて化学物質を流し込まれる。
陽圧、減圧を利用したかのような異様な隙間風や換気の逆流から
おそらく換気口や配電管が細工されている模様。

- 希釈した催涙ガス->皮膚や目、肛門が染みる。
- 酸素を減らした空気->高山病様症状、ボンベから出したてなのか異常にドライ。
- 酸素を足した空気->慢性的過呼吸状態。
- 自律神経に作用する薬物->思考困難、酩酊、脱力状態、呼吸が異様に浅くなる。

大まかにはこれくらい。

皮膚が負けて排便不良、頭痛でフラフラになる。
自分も大学教養程度の知識しかないためリストに確証は持てない。
量で効果が変わることもあるので複数あるようで一種類かもしれないし、
本命を隠すためにダミーを展開している可能性もある。
ただ、いずれもパニック狙いなので決して激高しないこと、倒れてでも無視。

自律神経の薬物については経口なのか、吸入なのか、皮膚付着なのか
経路特定できていない。
現行犯で捕まえようとしてトラブルを起こすと嵌められてしまうので注意。
薬効曲線、身体反応、持続時間等から経路を割り出し、
ある程度は割り切って、対応できるものだけこっそりと対策すべし。
僕の場合はマスクや保護クリームで多重対策を行った。

外出先で、車両使用、あるいは徒歩での気体散布もあるように見える。
外出中に不意に急激かつ異様な便意を催し困り果てたことがあった。
あと、便意といえば座面に制汗剤。

### ノイズ系
口笛、大音量での笑い声、ボールの音、クリック音、モスキート音
などを深夜早朝時間を問わず近隣、外出先で響かせる。
集合住宅の場合、隣室、上階から響くこともあるが、それを以って
部屋を押さえられていると考え踏み込むのは危険、抑えるべし。
挑発して近隣トラブルを起こさせ消耗させる狙いだろう。

### 複合系
上記の同時組み合わせ。
自律神経の薬で呼吸を浅くして酸素調整空気で選択的ノックダウンを狙う、
自律神経の薬剤使用と座面に制汗剤の組み合わせで脱糞、など。

### ハニートラップ
対象者の周囲に好意を装った異性を配置し
口説くようなことがあったらセクハラ被害を訴えて居場所をなくす。

※昭和の時代から続く伝統ある手法で元警官、元高砂市議の松本均も
30年ほど前に告発本で同一の被害を訴えている。

以上を駆使し精神病患者に仕立てたら定期的な繰り返しで
偽の精神病状態をキープする。
おとなしくしてても免除されず、半永久的にカモである。

ターゲットの選定基準についてはよくわからん。
単純にカモにしやすいかどうか、ということかも。
精神病院では、子供っぽい、孤立ぎみ、変人といった印象はあるものの
認識の異常を起こした精神病患者には見えない、この人カモられたな、
っていう人をちらほら見る。

実施機関は下で考察しているが多分警察。（あくまで多分）
地域住民や商業施設、交通機関の従業員など参加者の動員は幅広い。
ここまで幅広くやられると不特定多数、所謂「みんな」って奴になって
ますます精神病っぽさがでるのでタチが悪い。
それにつけこんで県管轄下にある防犯ボランティアから「みんな」的な
幅広い人員を動員していると思われる。
防犯ボランティアについては自治体の条例や警察の通知等を参照。


## 考察
あくまで考察で裏付けのない部分も多い。
参考にしてもらうのは問題ないが、真偽は各人で考察、確認してほしい。

### 実施機関
動員人員や機材の豊富さ、対応範囲の空間的、時間的広さからは県警、
もしくは自治体が想定される。
また、精神障害者として落とし込もうとする対応からも
警察庁と県警、厚労省と自治体の線で管轄への囲い込みと理解できる。

ただ、秘密保持の強さ、即応体制、展開力は警察でないと説明できない。
嫌がらせの強引さは問題発生時、自部署内での処理が前提のはずだ。

県警が主軸、自治体は厚労省管轄下での補佐ではないだろうか。
（で、後半で論ずる対策の対象は警察に決定。）

創価とか、会社が、、とかの記事をみることもあるが
嫌がらせの規模や内容がいち組織でできるスケールを超えている。
該当のいち組織が目立つ形で前衛をしているだけで警察主軸と考える。

### 目的
「そんなの妄想でしかないよ、警察も得しないし」
むかつくことに一定の説得力がある。

確かに警察が木っ端のような市民をいじめても意味がないようだが
少し内情を知っていると違うものが見えるてくる。

交通違反のノルマについて聞いたことはないだろうか。
あきらかにノルマのための取り締まりってあるけど、それに似た形で、
防犯のノルマがあって、事件数や検挙数等で予算、人員が決まるとしたら。
もっと言うと、非監置病者としてのターゲットリストがあってその数で。。。
他には、多数の民間人が関与するため謝礼金の効率的な裏金化も可能。
変人を非監置病者としてでっち上げるだけで組織が維持、拡充できる、
そういった打出の小槌なのではないだろうか。

※非監置病者：入院していない精神病患者
※裏金化：場合によっては厚労省管理予算から横断的に行われているかも。
追跡が困難になるためトンネルとして最適ではないか？

手先となる防犯ボランティアにもメリットがないわけではない。
防犯協力をする企業や市民団体に自治体から補助金がでるようだが
その査定は警察が関与しないと難しいだろう。
利益供与の関係が生まれるわけだが、それによってできたネットワークは
キックバックのやりとりや身内の問題もみ消しなど大いに役に立つ。

非監置病者のでっち上げで関係者内に資源が形成される、という寸法だ。
あくまで想像だけど。

### 精神科医
精神科医も多少なりと問題意識はあるはず！！
それでも加担する理由、あくまで調査からの想像だが。。。

ひとつ、弱いものいじめで県や国からお金をもらえるので。
切羽詰まっている病院やいじめ大好きマンが経営する病院もあるとは思う。

ふたつ、弱みを握られている。
精神障害関連の法律は厚労省管轄だが、警察の要望が反映されているそう。
それは当然のこと、、、通院継続中の患者が違法行為を行ったときに
警察が尻ふき対応するので親の厚労省が金玉を握られているわけだ。

みっつ、たちの悪い患者に同様の手法を使うことが黙認許可される。
面倒な患者を無力化できるのならばバッチコイ、なのだろうか。

実際ありえる話かと思う。

社会の要請に応えてるだけなのに医者扱いされないという愚痴を聞くが
行政府とつるんで脱法行為をしているのが実態なので。。。

話してみると個人レベルでは問題意識の存在を感じることもあるのに。。。

### 良識のある人は強硬に反対するでしょ？
警察内では反対は出ない、個人としてはともかく組織としては。
単純にそういう組織だから。
防犯ボランティアなど協力者の心理的抵抗は大きいと思う。
ただ、そんなものは簡単に打ち破れる、、、あくまで経験からの想像だが。

まず、ターゲットに変人（評判が悪い人）を選定する。
さらに先発（警察本体やその身内）があらかじめ挑発行為を行っておく。

ついで子供や女性の集団に身内を仕込んで嫌がらせをしかける。
全員仕込みではなく、集団の大部分を一般人にして
ターゲットにだけわかる先発で使った挑発方法を使用するのがポイント。
あとはうまく引っかけて、うわさが広がるのを待つだけ。

警察は市民を黙らせるには女性、子供を使うのが有効と知っている。
例えば、交通取り締まりの苦情にはワンパターンで子供の絡む事故の話を
持ち出して丸め込もうとする。

で、あいつは危険だ、子供が女性が、というと半分の人間は黙る。
残りの半分は、問題があったときの責任はどうするんだ、で黙る。
リスクを負う自己判断を避けて大勢に従う善人の傾向を逆手にとる訳だ。

対象者の身内を巻き込むときも同様の手口を使っているのだろう。
善人は基本的に自己判断に弱いので、でっち上げの証拠と責任論だけで
簡単に落ちてしまう。

違法行為の常として、おかしいと気づいた時には足抜けできない、、、
この筋書き通りだとは思わないが、だいたい似たようなものだろう。

### 国の決めたこと？
ではない。。。
国民の代表たる議員が国会で議決した内容に基づく行政行為のみ
国の決めたこと、とみなすことができる。

根拠となる法律がないので、役人が勝手にやっていること。
むしろ違法行為、脱法行為でもある。
対象への通告もない、実施基準もない、にもかかわらず強制的、
ということもあわせ行政行為の体をなしていない。

国家権力を濫用し組織的に不正を行っている、が正しい解釈。

一部の変人の迷惑行為が許せないにしても、法律を作って対応すれば
いいだけの話で、官公庁は法案を提言するルートも持っている。

法律に頼らず行政指導する手もありその場合任意協力で行えばよい。
それだけでも監視、抑制効果はあるし、
現行法でカバーする部分と連携させれば充分に機能する。

そうならないのは不正にうまみがある、ということ。

### 政府のすることだから支持しないと！
政府（国やその配下の官公庁）のすることは正しい、という考え方。
法的にはともかく目的は正しいはずだから邪魔するな、と。
無謬性、というやつだが、ありえない話であることは
直感的にも経験的にもわかると思う。警察の交通取締まりとか。。。

でも、正しいという前提じゃないと行政行為が成立しないので
法律で決めたルールに従って処理する場合に限り
無謬であると見做して話を進める、というのが法治の前提。

法律では処理できない問題が発生することは実情としてわかる。
その場合は優秀で熱意があり問題の本質を理解する担当者が、
違法に関する処罰への恐怖と問題解決への使命感の間で真剣に悩んで、
一回限りの緊急避難としての扱いで処理をするべきである。

おそらくその積み重ねが現状を作っているのだろうが
実情として資質的にふさわしくない人員によって前例踏襲的に
処罰の緊張感なく運用されているのではないだろうか。
そこには際限のない違法脱法行為の拡大、利益誘導などの腐敗しかない。

その場限りで関係者のために善処したことが、
祖先の努力を踏みにじり子孫の安寧を破壊することとなるのである。
決定に関われない外野の市民のそれも同様である。


## 対策
粗暴行為を起こさせて措置入院に追い込むためしつこく挑発されるが
権力の十八番の暴力で勝負するのは民間人には不可能。
また、薬物を使用し酩酊状態に追い込み保護を狙うケースは
留置所に放り込まれたらアウトである。
職質を受けたら協力し、尚且つ保護の要件を逃れるよう立ち回ること。

身柄を拘束できない程度であっても触法行為は
人員、機材強化の原因となるため注意。
リソース確保目的であろうか積極的に誘導、挑発してくることもある。

とにかく相手の得意なフィールドでは徹底的に逃げ回る。
相手の仕掛けてくる嫌がらせは実績のある完成されたトラップであり
それに乗るのは圧倒的に自分側不利と思っていい。

逃げ回りつつ相手の手口などを観察し、得られた情報を用いて
手間ばかりかかる割に合わない相手としての印象付けを狙っていく。
それには相手の弱点を突くことと自分の長所を活かすことが肝となる。

相手の弱点は、嫌がらせが違法脱法行為であり公になるとまずいこと、
カンバンが汚れることを極端に嫌うこと。
組織が大きいこともうまく付け込めば弱点として利用できる。

長所は個人に依存するので、僕個人のケースに限った話では
文献調査能力、事務能力、大きな組織内でその仕組みを見た経験、
以上となる。

それを元にしたのが別紙である。

ちょっとした問い合わせだけで相手組織が揺らぐわけもないが
内部で叱責や不信などを発生させ少しずつ人材、予算を削っていき
弱い部分に壊れていってもらうことが狙いである。

皆が使える一撃必殺の戦略が提示できないのが片手落ちだが、
相手が相手だけに、独自の作戦を立てて反撃してくる厄介な敵である、
と認識させてあきらめさせるぐらいしか手はないと思われる。

あと、常に相手の出方を観察し、資源の使用状況に目測をつけること。
相手の勢いに合わせて自分の資源を無駄なく投入し
成果を挙げさせず恥をかかせ気勢を殺いで、鼻歌交じりで戦うべし。


## 目標
おそらく、目標は直接的には客観的証明が不可能な方法論を以て
対象者をいつでも簡単に無力化できる状況を秘密裡に作り、
定期的に無力化を実行すること。
反抗的であると無力化に工数がかかると見做され苛酷になるが
おとなしくしても免れるわけではなく、必ず定期的に行われる。

経験では春と秋、要人訪問、国際イベントなどが機となるようだ。
予算の投下が関係してくるのであろうか。


## 分析
ここで相手組織、おそらく警察、について。

### 人間性軽視
日本で生きていると個人個人の価値観、良心を破棄させられ
人格の総入れ替えが行われるタイミングがある。
入社、部活動への加入などであるが警察はその最たるものであろう。
この副作用が他者の人間性の否定と、
ルールで処理できない事項の判断における良識の不在である。

人格の入れ替えをされた人間が他人の人格や人生を痛めつけることに
良心の呵責を覚えるとは考えにくい。

規則にないこと、その応用で解決できないことは人生経験を以て
良識的に判断するしかない。
民間人なら人生で培ってきた良識、哲学、美学で以て正しいことを
判断するところが入社時にそんな基準は捨て去っている。
ならば、わかりやすい基準は勝つほう、強いほうが正しい、である。

脱法が明白であっても、それがうまく秘密裡に運んでおり
組織を強化し、構成員に恩恵をもたらすのであれば反対する理はない。
良心の存在は忘れたほうが良さそうである。

※カルトの洗脳でもなし、そこまではないだろうと思ったならば
多少の良識が残っている人ほど、行為の悪辣さを理解しており
後戻りできない、ということを念頭に置いたほうがよい。

### 朕は国家なり？
「国家権力なめるなよ！？」
聞いたことはないであろうか。

警察を含む木っ端役人は自分や自分の組織を以て「国家」や「社会」を
名乗ることが多々ある。
法的には権力の執行を国家から限定的に委譲されているだけであるが
立場を使って容易に国家の名を隠れ蓑にできる現状がある。
繰り返し唱えるうちに言葉に飲み込まれるのだろうか、
自分たちこそが国家でありその思想に反する者は反社会性人格の持ち主、
ひいては精神障害者であり、管理が必要である、
自分たち国の本流を守るのだ、と大真面目に思っている節がある。

自分たちの邪魔になる人たちをたたく理由がついてしまうと
前項の人間性軽視と相まってこれが暴走して止まらない。
暴力行為、迫害行為を取り締まる法律は彼ら自身の管轄なのだから。

まあ、仮にこの通りの思想であれば確実に日本では「過激派」なので
日の元に晒せば干からびる他ないのが救いではある。

### 反撃
ここまできて注目すべきは方法論の無理である。
人間性を極端に無視した集団は、腐敗、分解、暴走のいずれかに走る、
そこが弱点であると見る。

倫理的、法的に問題がある行為が容易に行える理由、
実行者が自分の名と顔によって裁かれる可能性が極めて低く、
万が一のことがあっても組織が隠れ蓑になってくれるから、である。
が、強さのもとでもある匿名性、秘密性は弱点でもある。

まず、日の元に晒して、あるいは日の元に晒される恐怖を与えて
脆弱な人間性を干物にしてしまう手が考えられる。
外部からは匿名でも組織内ではそうではないことを利用し
特定部署、特定職への応力集中を狙う手もある。
質と量を稼げば無理のある行為の維持、あるいは組織そのものの
維持を困難にすることができる。

けん制する方法、攻撃する方法、防御する方法、いずれにおいても
上記を踏まえれば自分の可能な方法論で良いのである。
適法行為でそれを自分なりに構成できれば強力な武器となる。
目的、手段共に適法で、
相手が違法、不当な場合にそれに対し効力を発揮する仕掛けを構築し、
パナマ文書のようなの火種にあわせて大規模展開、が理想形。
